# Yumyard

Yumyard is a full-stack web application designed for food ordering and restaurant management. It provides both users and administrators with a seamless experience in navigating, ordering, managing, and engaging with the platform.

## Features

### For Users:
- **Ordering:** Browse and order from a variety of food items available on the platform.
- **Search:** Easily find specific dishes or cuisines using the search functionality.
- **Contact:** Connect with administrators via a user-friendly form for queries or assistance.
- **Feedback:** Provide feedback on the services and overall experience for continuous improvement.
- **Blogs:** Access informative blogs for culinary insights, recipes, and restaurant news.
- **Authentication:** Secure authentication through OTP verification ensures user privacy.
- **Maps:** Integration of map functionalities for locating restaurants and delivery addresses.
- **Payments:** Seamless payment gateway integration for secure online transactions.
- **Receipts:** Automatic generation of receipts for orders and donations.
- **Donations:** Facility for users to make donations to charitable causes through the platform.

### For Administrators:
- **Data Visualization:** Interactive charts provide insights into user activities, orders, and trends.
- **News Updates:** Regular updates keep users informed about promotions, events, and news.
- **Product Management:** Comprehensive tools for managing food items, prices, and availability.
- **User Management:** Efficient management of user accounts, profiles, and interactions.

## Getting Started

To run the Yumyard project locally, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/yumyard.git`
2. Navigate to the project directory: `cd yumyard`
3. Install dependencies: `npm install`
4. Start the server: `npm start`
5. Access the application at [http://localhost:3000](http://localhost:3000)
